This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
npm install

npm install zod

npm install react-hook-form

npm install @mui/material @emotion/react @emotion/styled

npm install next-auth@beta
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Thoughts
- Something i learnt while creating this project you shouldnt specify pages as use server because that means only server side logic can appear on that folder so no client side can be nested, so better to not declare that in the page and let it be default server side with client side components in it.
